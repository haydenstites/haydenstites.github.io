## TMBot
{{< paige/youtube video="6opxDsXKxTs">}}
<a style="font-size: 1.2rem;" href="https://github.com/haydenstites/TMBot">TMBot GitHub repository</a>
<p style="max-width: 80%; margin: 0 auto;"><i>TMBot is a Reinforcement Learning (RL) package for Nadeo's Trackmania 2020 written primarily in Python.
It is built off Stable Baselines 3 and aims to provide a painless interface for testing and developing new RL methods in customizable 3D environments.
The project was entirely concieved and produced by myself, the process of doing so taking around 2 months.
While other machine learning tools targeting Trackmania have been written, my approach is more applicable to general RL solutions,
giving the model more of what a real player would see and less hard info taken directly from the game engine.
Note that the model demo shown above was only trained for 300,000 steps, a small fraction of what many modern models are trained to, and represents only the lowest boundary of TMBot's usage.
